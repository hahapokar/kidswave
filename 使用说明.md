## 项目简介

KidsWave 是一个专为童装设计师打造的现代化作品集展示平台。它提供了优雅的界面设计、灵活的内容保护机制，以及便捷的后台管理功能。

**在线演示:** https://hahapokar.github.io/kidswave/

### 技术特点

- 🎨 基于 React + TypeScript 构建
- 📱 完全响应式设计
- ⚡ 使用 Vite 实现快速开发和构建
- 🚀 通过 GitHub Actions 自动部署
- 🔐 多层级内容保护机制

---

## 主要功能

### 1. 作品展示系统

- **智能筛选**: 按品类（外服/家居服/服饰/花稿）和可见性筛选
- **详细信息**: 每个作品包含标题、描述、价格和定制选项
- **价格计算器**: 自动计算基础价格和附加选项
- **图片保护**: 水印功能防止未授权使用

### 2. 三级可见性控制

#### 公开 (PUBLIC)
- 所有访客都可以查看
- 适合展示常规作品

#### 半公开 (SEMI_PUBLIC)
- 需要输入密码才能查看详情
- 适合会员专属内容
- **默认密码: 8888**

#### 专属定制 (EXCLUSIVE)
- 只展示预览，不提供下载
- 需要联系设计师解锁
- 适合品牌定制或高端作品

### 3. 管理员后台

- **密码保护**: 只有管理员可以访问
- **入口位置**: 
  - 顶部导航栏的"管理入口"按钮
  - 底部的"维护者登录入口"链接
- **默认密码: admin666**

## 管理员功能

### 访问后台

1. **点击"管理入口"按钮**
   - 位置：页面顶部导航栏

2. **输入管理员密码**
   - 默认密码：`admin666`
   - ⚠️ 部署前请务必修改！

3. **进入 Decap CMS 后台**
   - 可以管理网站内容
   - 支持可视化编辑

### 修改管理员密码

编辑 `App.tsx` 文件，找到第 57 行：

```typescript
if (adminPassword === 'admin666') {
  // 将 'admin666' 改为你的新密码
  // 例如：'MySecurePassword2024!'
  window.location.href = '/kidswave/admin/';
}
```

## 内容管理

### 添加新作品

编辑 `services/mockData.ts` 文件，在数组末尾添加：

```typescript
{
  id: '17',                                    // 新的唯一ID
  title: '春日花园系列',                        // 作品标题
  coverImage: 'https://images.unsplash.com/photo-xxx',  // 封面图片
  category: Category.OUTERWEAR,                // 品类
  ageGroup: AgeGroup.KIDS,                     // 年龄段
  visibility: Visibility.PUBLIC,               // 可见性
  basePrice: 699,                              // 基础价格
  description: '灵感源自春日花园的清新设计...',  // 作品描述
  addons: [                                    // 附加选项
    { label: '工艺版单 (B)', price: 120 },
    { label: '齐色建议 (C)', price: 60 }
  ]
}
```

### 品类说明

- `Category.OUTERWEAR` - 外服
- `Category.LOUNGEWEAR` - 家居服
- `Category.ACCESSORIES` - 服饰
- `Category.PATTERNS` - 花稿

### 年龄段说明

- `AgeGroup.INFANT` - 婴童 (0-3岁)
- `AgeGroup.KIDS` - 小中童 (4-12岁)

### 可见性设置

- `Visibility.PUBLIC` - 公开
- `Visibility.SEMI_PUBLIC` - 半公开（需要密码）
- `Visibility.EXCLUSIVE` - 专属定制

### 使用自己的图片

#### 选项1：使用图床（推荐）

推荐服务：
- **Unsplash**: https://unsplash.com/ （免费高质量图片）
- **Imgur**: https://imgur.com/ （免费图床）
- **Cloudinary**: https://cloudinary.com/ （专业CDN）

使用方法：
```typescript
coverImage: 'https://images.unsplash.com/photo-1234567890',
```

#### 选项2：存放在仓库中

```bash
# 1. 创建图片目录
mkdir -p public/images

# 2. 复制图片到该目录
cp 你的图片.jpg public/images/

# 3. 在代码中引用
coverImage: '/kidswave/images/你的图片.jpg',
```

⚠️ **注意**: GitHub 有 100MB 单文件限制，建议图片优化后上传。

## 安全设置

### 必须修改的密码

#### 1. 会员查看密码

在 `App.tsx` 第 48 行：

```typescript
if (viewerPassword === '8888') {
  // 改为你的密码，例如：
  if (viewerPassword === 'MyMember2024!') {
```

#### 2. 管理员密码

在 `App.tsx` 第 57 行：

```typescript
if (adminPassword === 'admin666') {
  // 改为你的密码，例如：
  if (adminPassword === 'AdminSecure2024!') {
```

### 密码强度建议

✅ **好的密码**:
- 至少 8 个字符
- 包含大小写字母
- 包含数字和特殊字符
- 例如：`KidsWave@2024!`

❌ **避免使用**:
- 纯数字（如：12345678）
- 常见单词（如：password）
- 个人信息（如：生日）

## 常见问题

### Q1: 部署后网站显示 404

**原因**: base 路径配置不正确

**解决方案**:

1. 检查 `vite.config.ts` 第 7 行：
   ```typescript
   base: '/kidswave/',  // 必须和仓库名一致
   ```

2. 如果仓库名不是 `kidswave`，需要修改：
   ```typescript
   base: '/你的仓库名/',
   ```

3. 同时修改 `App.tsx` 第 60 行的管理员跳转路径

### Q2: 管理员登录后跳转 404

**原因**: 管理员后台路径不正确

**解决方案**:

编辑 `App.tsx` 第 60 行：
```typescript
window.location.href = '/你的仓库名/admin/';
```

### Q3: 图片无法加载

**可能原因**:
1. 图片链接不是 HTTPS
2. 图片服务器不支持跨域
3. 图片路径错误

**解决方案**:
1. 使用 Unsplash 等支持 HTTPS 的图片源
2. 或将图片放在 `public/images` 目录
3. 检查路径是否包含正确的 base 路径

### Q4: 会员密码验证失败

**检查事项**:
1. 是否修改了 `App.tsx` 中的密码但忘记重新部署？
2. 输入的密码是否区分大小写？
3. 密码前后是否有空格？

### Q5: GitHub Actions 部署失败

**常见原因**:
1. 权限设置不正确
2. 依赖安装失败
3. 构建过程出错

**解决方案**:
1. 进入 Settings → Actions → General
2. 选择 "Read and write permissions"
3. 查看 Actions 标签页的错误日志
4. 本地运行 `npm run build` 测试

### Q6: 本地开发服务器无法启动

**解决步骤**:
```bash
# 1. 删除依赖
rm -rf node_modules package-lock.json

# 2. 清除缓存
npm cache clean --force

# 3. 重新安装
npm install

# 4. 启动服务器
npm run dev
```

### Q7: 如何更改网站标题和品牌信息？

**需要修改的文件**:

1. **index.html** - 页面标题
   ```html
   <title>KidsWave | Designer Portfolio</title>
   ```

2. **App.tsx** - 品牌信息
   ```typescript
   // 第 72 行左右
   <h1>KIDSWAVE</h1>
   
   // 第 85 行左右
   <button>管理入口</button>
   
   // 底部 Footer 部分
   "design@kidswave.studio"
   "杭州市滨江区创意路 88 号"
   ```

### Q8: 如何添加自定义域名？

**步骤**:

1. 在项目根目录创建 `public/CNAME` 文件：
   ```
   www.yourdomain.com
   ```

2. 在域名服务商处添加 CNAME 记录：
   ```
   类型: CNAME
   名称: www
   值: hahapokar.github.io
   ```

3. 推送代码，等待 DNS 生效（可能需要几小时）

4. 在 GitHub 仓库 Settings → Pages 中设置自定义域名
